Preciso implementar melhorias no sistema de acesso da área do aluno, incluindo criação e redefinição de senha.

Regras e funcionamento desejado:

1. Identificação do aluno:
- Cada aluno já possui cadastro na plataforma com CPF e/ou e-mail
- O login pode ser feito utilizando CPF ou e-mail

2. Primeiro acesso:
- O sistema deve permitir que o aluno realize o primeiro acesso criando uma nova senha
- Caso seja definida uma senha padrão inicial, o sistema deve obrigar a troca no primeiro login

3. Cadastro de nova senha:
- Deve existir a opção para o aluno cadastrar uma nova senha no primeiro acesso
- A senha deve ser de escolha do usuário e armazenada com segurança (criptografia/hash)

4. Alteração de senha:
- O aluno deve poder alterar sua senha a qualquer momento dentro da plataforma, na área logada

5. Recuperação de senha:
- Deve existir a opção “Esqueci minha senha” na tela de login
- Ao selecionar essa opção, o sistema deve enviar um link de redefinição para o e-mail cadastrado do aluno
- O link deve permitir que o aluno cadastre uma nova senha de forma segura

6. Interface:
- Tela de login simples com CPF ou e-mail + senha
- Opções visíveis:
  • “Criar nova senha” (para primeiro acesso)
  • “Esqueci minha senha”
- Fluxo intuitivo e responsivo (principalmente para celular)

7. Segurança:
- Garantir que cada CPF/e-mail esteja vinculado a apenas um aluno
- As senhas devem ser protegidas com criptografia
- Evitar acesso não autorizado

Objetivo:
Permitir que os alunos tenham acesso individual e seguro à plataforma, com autonomia para criar, alterar e redefinir suas senhas de forma simples e segura.

Caso necessário, posso fornecer a estrutura atual do cadastro de alunos para facilitar a implementação.
